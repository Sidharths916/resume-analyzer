Getting started

How to get your free API key

This app uses your own Anthropic key — so it's always free for us to run, and your data never touches our servers.

Step 01
Create a free account
Go to console.anthropic.com and sign up. No credit card needed to start.
Step 02
Go to API Keys
In the left sidebar click API Keys, then Create Key. Give it any name — "resumeai" works.
Step 03
Copy and paste it in
Copy the key (starts with sk-ant-), paste it into the API key field in the app, and hit Save. That's it.
New accounts get $5 free credit — that's roughly 300–500 resume rewrites. Each rewrite costs about $0.01. Your key is stored only in your browser session and deleted when you close the tab. We never see it.
